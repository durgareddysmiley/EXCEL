# Convert Excel Table to Normal Range (Quick Fix)

## Problem
When clicking a cell, Excel shows:
[@ColumnName]
instead of C2, C3.

## Reason
The data is in an Excel TABLE.

## Steps (Method A â€“ Best)
1. Click any cell inside the table
2. Press Ctrl + A (select entire table)
3. Go to Table Design tab
4. Click Convert to Range
5. Click Yes

## Result
- Table Design tab disappears
- Cell references become C2, C3, etc.

## Exam Tip
Avoid Excel Tables while learning VLOOKUP.
